AzureStorageApi

This is the AzureStorageApi API

getContainers

List the names of the available containers in this storage. Use 'include_properties' to include any properties of the containers.

URL

https://next.cloud.dreamfactory.com/rest/AzureStorage

HTTP Method

GET

Response Type

ContainersResponse

Parameters

    include_properties : boolean (query)
    optional

    Return any properties of the container in the response.

createContainers

Post data should be a single container definition or an array of container definitions. Alternatively, override the HTTP Method to pass containers to other actions.

URL

https://next.cloud.dreamfactory.com/rest/AzureStorage

HTTP Method

POST

Response Type

ContainersResponse

Parameters

    body :  
    ContainersRequest (body)

    Array of containers to create.

    check_exist : boolean (query)
    optional

    If true, the request fails when the container to create already exists.

    X-HTTP-METHOD : string (header)
    optional

    Override request using POST to tunnel other http request, such as DELETE.

deleteContainers

Pass a comma-delimited list of container names to delete. Set 'force' to true to delete all containers. Alternatively, to delete by container records or a large list of names, use the POST request with X-HTTP-METHOD = DELETE header and post containers.

URL

https://next.cloud.dreamfactory.com/rest/AzureStorage

HTTP Method

DELETE

Response Type

ContainersResponse

Parameters

    names : string (query)
    optional

    List of containers to delete.

    force : boolean (query)
    optional

    Set force to true to delete all containers, otherwise 'names' parameter is required.

URL

https://next.cloud.dreamfactory.com/rest/AzureStorage/{container}/

HTTP Method

GET

Response Type

ContainerResponse

Parameters

    container : string (path)

    The name of the container from which you want to retrieve contents.

    include_properties : boolean (query)
    optional

    Include any properties of the container in the response.

    include_folders : boolean (query)
    optional

    Include folders in the returned listing.

    include_files : boolean (query)
    optional

    Include files in the returned listing.

    full_tree : boolean (query)
    optional

    List the contents of all sub-folders as well.

    zip : boolean (query)
    optional

    Return the content of the folder as a zip file.

deleteContainer

Set 'content_only' to true to delete the folders and files contained, but not the container. Set 'force' to true to delete a non-empty container. Alternatively, to delete by a listing of folders and files, use the POST request with X-HTTP-METHOD = DELETE header and post listing.

URL

https://next.cloud.dreamfactory.com/rest/AzureStorage/{container}/

HTTP Method

DELETE

Response Type

ContainerResponse

Parameters

    container : string (path)

    The name of the container you want to delete from.

    force : boolean (query)
    optional

    Set to true to force delete on a non-empty container.

    content_only : boolean (query)
    optional

    Set to true to only delete the content of the container.

getFile

By default, the file is streamed to the browser. Use the 'download' parameter to prompt for download. Use the 'include_properties' parameter (optionally add 'content' to include base64 content) to list properties of the file.

URL

https://next.cloud.dreamfactory.com/rest/AzureStorage/{container}/{file_path}

HTTP Method

GET

Response Type

FileResponse

Parameters

    container : string (path)

    Name of the container where the file exists.

    file_path : string (path)

    Path and name of the file to retrieve.

    include_properties : boolean (query)
    optional

    Return properties of the file.

    content : boolean (query)
    optional

    Return the content as base64 of the file, only applies when 'include_properties' is true.

    download : boolean (query)
    optional

    Prompt the user to download the file from the browser.
